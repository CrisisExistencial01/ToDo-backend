# Simple ToDo app

## Installation
### Clone the repos
``` bash
git clone https://github.com/CrisisExistencial01/ToDo-backend
git clone https://github.com/CrisisExistencial01/ToDo-frontend
``` 
### Install dependencies (ToDo-backend)
```
npm install
```
### Install dependencies (ToDo-frontend)
```
npm install
```
## Run backend
```
npm start
```
## Run frontend
```
npm run dev
```
